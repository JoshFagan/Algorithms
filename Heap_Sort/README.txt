Instructions for running heap_sort.cpp:
execute command: make
execute command: ./heap_sort seed size

where seed is the seed to prime the random number generator
where size is the number of elements to sort

The output of heap_sort.cpp is automatically suppressed, if you wish to see the
unsorted and sorted arrays generated by heap_sort, uncomment lines 157 and 160
in heap_sort.cpp

Instructions for running inline_heap_sort.cpp:
execute command: vi Makefile
execute command: :1,$s/heap/inline_heap/g
execute command: :wq
execute command: make
execute command: ./hs seed size

where seed is the seed to prime the random number generator
where size is the number of elements to sort

The output of inline_heap_sort.cpp is automatically suppressed, if you wish to
see the unsorted and sorted arrays generated by heap_sort, uncomment lines 
78-82 and 110-114 in inline_heap_sort.cpp
